Business Model Canvas

Aplikasi Pengelola Keuangan: SmartSpend AI

🤝 Key Partners

  • Supabase: Manajemen user & Auth
  • NeonDB: Serverless PostgreSQL database
  • Penyedia API OCR: Untuk ekstraksi data dari foto struk
  • Komunitas Kampus: Partner promosi ke mahasiswa

⚡ Key Activities

  • Pengembangan fitur OCR struk
  • Optimasi sistem export PDF
  • Maintenance sinkronisasi database

💡 Value Propositions

MAIN USP
  • Input Instan: Catat pengeluaran hanya dengan foto struk (AI OCR)
  • Reporting Pro: Export laporan PDF rapi untuk evaluasi bulanan
  • Financial Literacy: Membantu mahasiswa/umum mengatur budget agar tidak "kritis" di akhir bulan

❤️ Customer Relationships

  • Freemium Model: Coba gratis, bayar untuk fitur pro
  • Support: Bantuan teknis via in-app chat
  • Feedback Loop: Update fitur berdasarkan request user

👥 Customer Segments

  • Mahasiswa: Yang ingin mengontrol uang saku & biaya kos
  • Penduduk Umum: Orang tua/pekerja yang ingin lebih hemat
  • Freelancer: Yang butuh laporan pengeluaran sederhana (PDF)

🛠️ Key Resources

  • Source Code Flutter
  • Cloud Infrastructure (Supabase/Neon)
  • Waktu & Keahlian Tim Developer

📱 Channels

  • Play Store & App Store
  • Social Media (TikTok/IG) untuk edukasi finansial
  • Word of Mouth (Rekomendasi teman)

💰z Cost Structure

  • Biaya Waktu: Pengembangan & riset tim
  • Operational: Langganan API OCR (per scan)
  • Scalability: Upgrade plan Supabase/Neon jika user melonjak

💵 Revenue Streams

  • Premium Subscription: Akses unlimited PDF Export
  • One-time Purchase: Buka fitur OCR selamanya
  • Ad-supported: Iklan banner minimalis di versi gratis
